				stage-1 : Unit Testing 			- using junit by developer.
				stage-2 : Integretion Testing 	- using junit by developer.
				stage-3 : Smoke Testing 		- by Tester
				stage-4 : Functional Testing 	- by Tester
				stage-5 : Regression Testing 	- by Tester
				stage-6 : Performance Testing 	- by Tester
